File Poliglotti ed Extra Dati

I formati dei file grafici ignorano tutto ciò che viene scritto dopo il loro marcatore di fine struttura (IEND nelle PNG).

L'immagine qui sotto nasconde un intero payload di testo iniettato in coda ai suoi byte binari archiviati.

Esegui un'ispezione della struttura grezza del file per trovare la stringa, estrai l'istruzione e digitala in console per generare la password dinamica!
